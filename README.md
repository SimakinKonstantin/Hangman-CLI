# Консольная версия игры "Висeлица"
Игрок пытается угадать загаданное слово, вводя буквы по одной за раз. Слово выбирается по уровню сложности, случайно из предварительно заданного списка слов и категории. Количество попыток ограничено, и за каждую неверную догадку визуализируется часть виселицы и фигурки висельника. Угадываемые слова берутся из JSON-файла либо из дефолтного json-файла по умолчанию.
Структура файла с описанием слов для игры:
```json
{
  "Категория1": [
    {"word": "Слово1", "hint": "Подсказка к слову"},
    {"word": "Слово2", "hint": "Подсказка к слову"}
  ],
  "Категория2": [
    {"word": "Слово3", "hint": "Подсказка к слову"},
    {"word": "Слово4", "hint": "Подсказка к слову"}
  ],
  "Категория3": [
    {"word": "Слово5", "hint": "Подсказка к слову"},
    {"word": "Слово6", "hint": "Подсказка к слову"}
  ]
}
```

## Функционал
* Программа выбирает случайную категорию слов, если этого не сделал пользователь.
* Программа выбирает уровень сложности, если этого не сделал пользователь.
* Программа выбирает случайное слово из заранее определенного списка слов.
* Игрок вводит одну букву за раз, чтобы угадать слово.
* Экран обновляется после каждого ввода, показывая уже угаданные буквы и прочерки на местах неуказанных букв.
* Визуализация виселицы добавляется постепенно с каждой неправильной попыткой.
* Игра завершается, когда слово угадано полностью или когда висельник полностью нарисован.
* Количество попыток ограничено и должно быть указано в начале игры.
* Для слова из каждой категории можно получить подсказку.

## Инструкция по запуску
Выполнить: `go run .\cmd\run\main.go`
